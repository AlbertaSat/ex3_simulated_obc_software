Command: cmdSeq:CmdSequencer

Mnemonic ID Description Arguments
Argument Type Description
CS_RUN 1536 (0x600) Run a command sequence file
fileName Fw::CmdStringArg The name of the sequence file
block Svc::CmdSequencer_BlockState Return command status when complete or not
CS_VALIDATE 1537 (0x601) Validate a command sequence file
fileName Fw::CmdStringArg The name of the sequence file
CS_CANCEL 1538 (0x602) Cancel a command sequence
CS_START 1539 (0x603) Start running a command sequence
CS_STEP 1540 (0x604) Perform one step in a command sequence. Valid only if CmdSequencer is in MANUAL run mode.
CS_AUTO 1541 (0x605) Set the run mode to AUTO.
CS_MANUAL 1542 (0x606) Set the run mode to MANUAL.
CS_JOIN_WAIT 1543 (0x607) Wait for sequences that are running to finish. Allow user to run multiple seq files in SEQ_NO_BLOCK mode then wait for them to finish before allowing more seq run request.
